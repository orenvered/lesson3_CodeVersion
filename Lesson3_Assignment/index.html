Lesson 3 Highlights

Oren Vered

Highlight #1 Version Control

The task of keeping a software system made up of many types and configurations organized and well functional for all project files. Users are able to make changes and upload them in one repository, which provides an elegant way of dealing with file sharing.

Highlight #2 Version Control System

A version control system, or VCS, is a repository of files, both local and remote. It allows for a user to edit files on their personal computer (local repository), which may then be uploaded to a remote repository, located online, which team members are able to view.

Highlight #3 Git

Git is a version control system which functions with local and remote repositories, and provides users with effective speed and proficiency to handle a project of any magnitude.
